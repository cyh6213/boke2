# RAG问答系统项目概述

## 项目简介

RAG问答系统是一个基于检索增强生成（Retrieval-Augmented Generation）技术的智能问答平台。该系统能够将用户问题与知识库中的文档进行智能匹配，结合大语言模型生成精准、上下文相关的回答。

## 核心功能模块

### 📚 文件上传与解析
支持多种格式文档的上传和解析：
- PDF 文档解析
- Word 文档解析  
- Markdown 文件支持
- 文本文件处理

### 🔍 文本处理与混合检索
- 文档向量化处理
- 语义检索算法
- 混合检索策略（关键词+语义）
- 上下文感知检索

### 💬 LLM交互
- 与大语言模型的交互接口
- 支持多种主流大模型
- 提示词工程优化
- 响应格式标准化

### 🔐 权限/登陆控制
- 用户认证系统
- 角色权限管理
- 文档访问权限控制
- Session 管理

### 🧠 记忆功能
- 对话历史记忆
- 上下文理解
- 多轮对话支持

## 项目迭代历程

### 第一次改进

- 引入混合检索策略
- 提升回答准确性

### 第三次改进
- 增加记忆功能
- 性能优化

## 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                    用户层                              │
│   Web UI / API / WebSocket                            │
├─────────────────────────────────────────────────────────┤
│                    业务层                              │
```

## 技术栈

| 分类 | 技术 |
|------|------|
| 后端框架 | Spring Boot |
| 数据库 | MySQL + Redis |
| 向量数据库 | Milvus/Chroma |
| 实时通信 | WebSocket |
| 日志系统 | 日志注解 |
| 性能监控 | 监控系统 + 压力测试 |

## 技术亮点

✅ **精准检索** - 支持多种检索策略，精准匹配知识库

✅ **实时交互** - WebSocket实现实时问答体验

✅ **记忆功能** - 对话历史记忆，支持上下文理解

✅ **性能优化** - 多级缓存策略保证系统响应速度

✅ **扩展性强** - 模块化设计，易于扩展新功能

## 应用场景

- **企业知识库问答** - 员工自助查询企业文档
- **智能客服系统** - 自动化客户服务
- **教育辅导系统** - 智能答疑解惑
- **个人知识管理** - 个人知识库检索

## 项目价值

RAG问答系统通过将检索技术与大语言模型相结合，解决了传统问答系统的知识时效性和准确性问题，为企业和个人提供了高效、智能的知识检索和问答体验。
